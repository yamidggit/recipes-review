* Application to keep track of user's recipes*

* Ruby version
    ruby 2.3.0p0

* Rails version
    ruby 2.3.0p0

* Configuration
    bundle install
    
* Database creation
    rake db:create
    
* Database initialization
    rake db:migrate
    
* How to run the test suite
    guard
    spring rspec (much faster)
    
*Stop spring
    spring stop
    
*Start the server
    rails server -b $IP -p $PORT
